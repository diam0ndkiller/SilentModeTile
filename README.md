# Silent Mode Tile
This is a small Android app that lets you create a Quick Setting Tile to cycle through Do Not Disturb Modes like DC7 Software's Silent Mode Toggle Widget did.

## DISCLAIMER
This projects was created using AI, so no license is being applied.

## Setup
In order to use the tile, go to *Settings > Apps > Special app access > Modes access* an enable access for `Silent Mode Tile`.
